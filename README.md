## Quickstart
Here's how you can get up and running with this project.

1. Clone this repository
```
git clone https://github.com/lcorr8/qarma.git
```

2. Install dependencies
```
bundle install
```

3. Migrate database models
```
rake db:migrate
```

4. Run project locally
```
rails s
```
